AP Computer Science A Video Series

Watch InterSTEM's AP Computer Science A crash course videos that are being released biweekly. Taught by Ronak Wani, an experienced programmmer and a Google Code national finalist, you will get to learn about the Java programming language and other materials covered in APCSA curriculum.



Module 01: Expressions and Operators. In this first lecture of the AP CSA series, Ronak covers arithmetic, logical and relational operators in Java programming language.



Module 02: Relational, Logical, and Compound Assignment Operators. This video cover various ways to compare variables and values through logic gates and mathematical inequalities.



Module 03: Comment statements in Java. Learn about different ways you can write comments in Java.



Module 04: Decision Making in Java. Ronak explains conditional statements and branching in Java.
